What's next?

I'd like to put a tutorial up, in case anyone wants to try this project themselves.

And if I could change one thing about Mano itself, it might be swap out the baseboard and use a regular Arduino Uno instead. Currently, Mano uses the Makeblock baseboard, which has the same ATmega32U4 processor as Arduino Leonardo. This was a great baseboard to get started with and learn the basics of controlling Mano. However, it doesn't have any customizable I/O aside from pre-fabricated Makeblock sensors. I'm itching to try out the chip I got for color- and gesture-sensing! I learned to solder in the hopes that I can use this chip in the future, along with a regular Arduino Uno, to train Mano to follow a certain color: https://www.sparkfun.com/products/12787.
