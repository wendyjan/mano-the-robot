An Overview

I made Mano using a Makeblock Starter Robot Kit IR Version, which includes an Arduino Leonardo/Makeblock Me Baseboard and various sensors. It is available at http://www.robotshop.com/en/makeblock-starter-robot-kit-v2-electronics-blue.html .

The board

The Makeblock Me Baseboard is essentially the Arduino Uno, with the same ATmega32U4 processor. However you can see that instead of GPIO pins to allow almost any sensors or other peripherals to be attached, there are 8 RJ25 ports. These ports provide simple access to any Makeblock peripheral module. For Mano, I used the ultrasonic sensor. I also used the infrared reciever module to keep the remote-control functionality that he came with out of the box.

The ultrasonic sensor

This is a closeup of the ultrasonic sensor, which looks like a pair of eyes! This sensor provides distance measurements to the processor. This is what allowed me to implement gesture-based motion control.

The infrared sensor

Here is Mano's infrared (IR) sensor. This is not directly part of my project, because Mano already came with remote control IR functionality. However I saw no reason to take that away!

The Makeblock Starter Robot Kit

This is the kit I used to build Mano. It's a great way to get started if you've never built much electronic stuff! Plus it allows for lots of customization. I would really recommend it, you can check it out at http://www.robotshop.com/en/makeblock-starter-robot-kit-v2-electronics-blue.html .
